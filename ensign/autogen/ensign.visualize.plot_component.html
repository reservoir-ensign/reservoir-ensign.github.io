ensign.visualize.plot_component¶

plot_component(decomp, comp_id, inline=True)[source]¶

Visualizes a specific component of a decomposition using blue-line charts.

Parameters
decompCPDecomp

Decomposition object from which to visualize a component.

comp_idint

The ID of the component to visualize.

inlinebool

Set to True if using within a Jupyter notebook and False to get PNG file as output. The output file is named ‘_comp_<comp_id>.png’.

Returns
figmatplotlib.figure.Figure

Figure object representing visualization.
